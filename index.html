Лабораторные работы по Git

6 работ по 40–50 минут каждая. Для каждой работы есть теоретическая справка, пошаговая инструкция, самостоятельное задание и контрольные вопросы.

  • Лабораторная работа №1. Основы: init, add, commit, status, log, diff
  • Лабораторная работа №2. Ветвление и слияние
  • Лабораторная работа №3. Удалённые репозитории (GitHub)
  • Лабораторная работа №4. Коррекция истории: amend, reset, revert, stash
  • Лабораторная работа №5. Модель ветвления Git Flow
  • Лабораторная работа №6. Интерактивный rebase, cherry-pick, теги, reflog

Перед началом работы

git config --global user.name "Ваше Имя"
git config --global user.email "ваш_email@example.com"

Критерии оценки

  • Выполнение всех шагов (4 балла)
  • Самостоятельное задание (3 балла)
  • Ответы на контрольные вопросы (3 балла)
  • Аккуратность и своевременность (1 балл)
