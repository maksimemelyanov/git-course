## Введение

### Что такое Git?
Git — распределённая система контроля версий (VCS), позволяющая отслеживать изменения в файлах, работать в команде, возвращаться к старым версиям, экспериментировать в ветках.

### Установка
- **Windows:** скачать Git for Windows (Git Bash)
- **macOS:** `brew install git` или скачать установщик
- **Linux (Ubuntu/Debian):** `sudo apt install git`
