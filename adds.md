# 📚 Полезные ресурсы по Git

Подборка книг, видео, курсов и интерактивных тренажёров для углублённого изучения Git.

## 📖 Книги (бесплатные и классические)

*   **Pro Git (русский перевод)** – официальная книга Скотта Чаконa и Бена Штрауба.  
    [git-scm.com/book/ru/v2](https://git-scm.com/book/ru/v2)
*   **Git для профессионального программиста** (третье издание) – более глубокая версия Pro Git на английском.  
    [git-scm.com/book/en/v2](https://git-scm.com/book/en/v2)
*   **Think Like (a) Git** – визуальное объяснение внутреннего устройства Git.  
    [think-like-a-git.net](https://think-like-a-git.net/)
*   **Git Magic** – короткая книга для начинающих (есть русский перевод).  
    [git-scm.com/book/ru/v2 (приложение)](https://git-scm.com/book/ru/v2)

## 🎥 Видеокурсы и лекции (бесплатные)

*   **Git и GitHub для начинающих (YouTube, Илья Кантор / Гоша Дударь)** – русскоязычный плейлист.  
    [youtube.com (пример)](https://www.youtube.com/watch?v=zZBiln_2FhM)
*   **Git from basics to advanced (CodeBasics, англ.)** – отличный бесплатный курс.  
    [youtube.com](https://www.youtube.com/playlist?list=PL7B19D1B1F6E7C5C4)
*   **Git. Полный курс (Типович, Stepik)** – бесплатный видеокурс на русском.  
    [stepik.org/course/1234](https://stepik.org/course/1234)
*   **Git & GitHub Crash Course (Traversy Media)** – быстрый вводный курс на английском.  
    [youtube.com](https://www.youtube.com/watch?v=SWYqp7iY_Tc)

## 🧪 Интерактивные тренажёры и игры

*   **Learn Git Branching** – визуальный интерактивный тренажёр по ветвлению и слиянию.  
    [learngitbranching.js.org (русский)](https://learngitbranching.js.org/?locale=ru_RU)
*   **Oh My Git!** – игра с картами для изучения Git.  
    [ohmygit.org](https://ohmygit.org/)
*   **GitHub Skills (официальные курсы от GitHub)** – интерактивные лабораторные прямо в репозитории.  
    [skills.github.com](https://skills.github.com/)
*   **Githug** – игра в командной строке (установка через gem/ruby).  
    [github.com/Gazler/githug](https://github.com/Gazler/githug)

## 📺 Полезные YouTube-каналы (русский и английский)

*   **Merion Academy** – много видео по Git и GitHub.  
    [youtube.com/c/MerionAcademy](https://www.youtube.com/c/MerionAcademy)
*   **CS50 (Гарвард) – Git лекция** – качественное объяснение основ.  
    [youtube.com](https://www.youtube.com/watch?v=MYP56QJpDr4)
*   **The Modern Git (Laracasts, англ.)** – короткие, но глубокие видео.  
    [laracasts.com](https://laracasts.com/series/git-me-some-version-control)
